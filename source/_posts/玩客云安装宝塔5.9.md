# 玩客云安装宝塔5.9

## 开始安装

<!--more-->

通过SSH登录服务器，输入

```
wget -O install.sh http://download.bt.cn/install/install-ubuntu.sh && bash install.sh
```

如果你是第一次安装，会报Pillow installation failed

解决方法：`sudo apt-get install python-setuptools`

然后再安装，你就能安装成功了~
